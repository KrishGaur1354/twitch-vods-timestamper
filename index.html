Getting Started

Twitch VOD Timestamp is a command-line tool for creating, managing, and sharing timestamps from Twitch VODs.

Basic Commands

twitch@timestamp:~$
timestamp add "Description" --vod VODID
Adds a timestamp at the current position
timestamp list
Lists all timestamps for the current VOD
timestamp export --format FORMAT
Exports timestamps to specified format (markdown, json, csv)

Configuration

Create a .timestamprc file in your home directory to customize default behavior:

{
  "defaultFormat": "markdown",
  "autoExport": true,
  "exportPath": "~/Documents/timestamps/"
}

API Integration

For developers wanting to integrate with the Timestamp API:

timestamp api-key generate
Generates a new API key for external applications

Frequently Asked Questions

Q: Does this work with live streams or only VODs?

A: The tool works primarily with VODs, but you can use the --live flag to timestamp live streams which will be converted to VOD timestamps when the stream ends.

Q: Can I share timestamps directly to social media?

A: Yes! Use the --share flag followed by the platform name:

timestamp share --platform discord
Discord-formatted link copied to clipboard!

Q: Is there a limit to how many timestamps I can create?

A: No, there is no limit to the number of timestamps you can create per VOD.

Q: Does this work on all operating systems?

A: Yes, we support Windows, macOS, and Linux.

Q: Can I categorize or tag my timestamps?

A: Absolutely! Use the --tag flag to add categories:

timestamp add "Epic play" --tag highlight,gameplay
Timestamp added with tags: highlight, gameplay
